# BaseU项目启动问题修复记录

## 日期：2025-11-04

## 问题描述
项目在启动时出现以下错误：
```
java.lang.IllegalArgumentException: Invalid value type for attribute 'factoryBeanObjectType': java.lang.String
```

## 根本原因
MyBatis Plus 3.5.4 与 Spring Boot 3.2.0 存在兼容性问题。

## 解决方案

### 1. 升级MyBatis Plus版本
- **从**: 3.5.4  
- **到**: 3.5.5

### 2. 降级Spring Boot版本  
- **从**: 3.2.0  
- **到**: 3.1.5

## 修改的文件

### pom.xml
```xml
<!-- Spring Boot 版本降级 -->
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.1.5</version>  <!-- 从3.2.0改为3.1.5 -->
    <relativePath/>
</parent>

<!-- MyBatis Plus 版本升级 -->
<properties>
    <mybatis-plus.version>3.5.5</mybatis-plus.version>  <!-- 从3.5.4改为3.5.5 -->
</properties>
```

## 启动命令

### Windows (PowerShell)
```powershell
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

或者使用提供的批处理脚本：
```powershell
.\start.bat
```

### Linux/Mac
```bash
./start.sh dev
```

## 注意事项

1. 确保MySQL服务已启动并监听在3306端口
2. 确保Redis服务已启动并监听在6379端口（如果使用Redis功能）
3. 首次启动会自动创建数据库
4. 默认使用dev配置文件，数据库用户名密码为root/666666
